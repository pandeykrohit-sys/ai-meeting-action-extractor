# AI Meeting Action Item Extractor

An AI-powered workflow built using n8n that automatically extracts action items from meeting notes and sends them via email.

## 🚀 Live Workflow

This project converts unstructured meeting notes into structured outputs including:
- Action items
- Owner names
- Deadlines

## 🔧 Tech Stack

- n8n (workflow automation)
- OpenRouter API (AI model)
- Gmail API (email automation)

## ✅ Features

- Input meeting notes manually
- AI extracts structured action items
- Formats output into readable text
- Automatically sends results via email

## ❗ Problem

Manually extracting action items from meeting notes is time-consuming and inefficient.

## ✅ Solution

This project automates the process using AI and workflows, saving time and improving accuracy.
